XHAgentOS 项目目录结构

XHAgentOS (项目根目录)
    |   app.py (入口：路由注册、服务启动、种子数据)
    |
    +---app (MVC 业务代码)
    |   |   __init__.py
    |   |
    |   +---controllers (控制层)
    |   |       auth.py       (登录/退出 Handler)
    |   |       base.py       (BaseHandler 基类：Cookie/认证)
    |   |       home.py       (AdminHandler：后台主页占位)
    |   |       admin.py      (用户管理 CRUD Handler 集群)
    |   |       __init__.py
    |   |
    |   +---models (模型层)
    |   |       db.py         (数据库连接、建表)
    |   |       user.py       (UserRepository：用户 CRUD + 分页)
    |   |       __init__.py
    |   |
    |   +---static (静态资源)
    |   |   +---css (样式文件)
    |   |   |       base.css  (基础样式，保留)
    |   |   |       login.css (登录页沉浸式样式)
    |   |   +---js (脚本)
    |   |   |       base.js
    |   |   +---lib (前端库)
    |   |   |   +---layui (LayUI v2.13.7：后台管理 UI)
    |   |   |           layui.js
    |   |   |           css/layui.css
    |   |   |           font/iconfont.*
    |   |   +---dist (下载的 ZIP 压缩包)
    |   |
    |   +---templates (视图层)
    |           login.html            (沉浸式响应式登录页)
    |           admin.html            (LayUI 后台管理布局：左侧菜单+右侧iframe)
    |           admin_dashboard.html  (后台主页占位)
    |           admin_user_list.html  (用户管理列表页)
    |           admin_user_form.html  (用户新增/编辑表单)
    |
    +---database (数据库)
    |       app.db (SQLite 数据库文件)
    |
    +---docs (开发文档)
    |       basePrompt.md (项目基础信息)
    |       codingPrompt.md (开发任务提示)
    |       fraework.txt (目录结构)
    |       requirementsPrompt.md (需求文档)
    |
    +---venv (虚拟环境)
