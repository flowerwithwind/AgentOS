# 提示词记录 — 模板

> **用途**：记录团队成员在开发过程中向AI Agent（如Trae、Cursor等）输入的提示词，作为考核资料提交。
> **提交要求**：每组一份提示词记录文件，打包提交。

---

## 使用说明

1. 每位成员复制此文件，命名为 `提示词_姓名.md`（如 `提示词_向党.md`）
2. 每次向AI Agent输入关键提示词时，按以下格式记录
3. 提交前汇总到小组提示词记录文件
