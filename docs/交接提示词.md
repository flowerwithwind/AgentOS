## 八、技术要点

### API 规范
```json
// 统一响应格式
{"code": 0, "message": "success", "data": {...}}
{"code": 1, "message": "错误信息", "data": null}
```

### 认证机制
- Cookie-based：`self.set_secure_cookie("user", username)`
- XSRF：`getCookie('_xsrf')` + `X-Xsrftoken` 请求头
- 装饰器：`@tornado.web.authenticated`

### 数据库访问模式
```python
# 示例：Repository 模式
from app.models.db import get_connection

def list_page(self, page, page_size):
    with get_connection() as conn:
        data = conn.execute("SELECT ... LIMIT ? OFFSET ?", ...).fetchall()
        total = conn.execute("SELECT COUNT(*) FROM ...").fetchone()["cnt"]
        return data, total
```

### 前端模板模式
```html
<body style="background: #f5f6fa;">
<div class="page-title">
    <i class="layui-icon layui-icon-xxx"></i> 页面标题
</div>
<!-- layui-table -->
```

## 九、对接流程（给组员）
